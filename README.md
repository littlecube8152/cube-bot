# Cube Bot
A bot based on pycord.


## Installation

Requires Python 3.10.  

1. Create a python virtual environment:
   ```bash
   python -m venv .
   source bin/activate
   ```
2. Install required packages:
   ```
   pip install -r requirements.txt
   ```
3. Add token via `config.json`:
   ```json
   {
      "token": "(your token here)"
   }
   ```
4. Run the bot.
   ```
   python main.py
   ```
